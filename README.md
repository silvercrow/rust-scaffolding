# Rust Scaffolding
> Rust,vscode


## Installation and usage
> edit the makefile to update your project_name and launch generator

```sh
make init
```

> launch rust:

```sh
make dev
```

> open vscode and start coding on your main file in {projectname}/src/main.rs:
